# 코딩 테스트 연습

## 커밋 규칙

| 이모티콘 | 커밋 유형    | 설명                    |
| -------- | ------------ | ----------------------- |
| ✨       | `feat`       | 새로운 문제 풀이 추가.   |
| 🐛       | `fix`        | 오류 수정.              |
| ♻️       | `refactor`   | 코드 리팩터링.          |
| 🧪       | `test`       | 테스트 케이스 추가.      |
| 📝       | `docs`       | 설명이나 주석 추가.      |

## 커밋 메시지 템플릿

- `[이모티콘] feat: 문제 이름` – 예시: ✨ feat: 두 수의 합
- `[이모티콘] fix: 문제 이름` – 예시: 🐛 fix: 최대공약수 오류 수정
- `[이모티콘] refactor: 문제 이름` – 예시: ♻️ refactor: 중복 제거

### 이름 형식

- `'[사이트 명] 문제 이름'`
  - 예시: `[LeetCode] Two Sum`
  - 예시: `[Programmers] 두 수의 합`
